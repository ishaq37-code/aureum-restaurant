# 👑 Aureum — Luxury Fine Dining Restaurant Landing Page

A premium, highly interactive fine dining restaurant landing page built with **React**, **Vite**, and **Tailwind CSS**. It features a dark-luxury theme, real food photography, smooth animations, counters, dynamic menu tabs, and a gorgeous responsive UI.

---

## 🤮️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS + PostCSS
- **Icons:** Lucide React

---

## 🚀 Quick Start (Local Run)

To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

### 1. Extract / Download files
Download the project files from this repository (essentially the contents of `/aureum-project`) or copy this repo locally.

### 2. Install dependencies
navigate to your project folder in your terminal and run:
```bash
npm install
```

### 3. Run development server
Start Vite's local dev server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser. ✨

### 4. Build for Production
To generate optimized production assets under the `dist/ directory:
```bash
npm run build
```

## 🌍 Deploying to Vercel (One-Click, Free)

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
2. Click **Add New** → **Project**.
3. Import your `aureum-restaurant` repository.
4. Click **Deploy**. Vercel will automatically detect Vite + Tailwind and configure the build settings. Búm! 🎅

Example live URL:  `https://aureum-restaurant.vercel.app`
